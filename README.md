# 小区停车场管理系统 java289

## 项目概述

小区停车场管理系统是一款基于SpringBoot和Vue的前后端分离的web网站应用，主要用于实现小区停车场的智能化管理。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- Windows

### 开发环境
- IDEA
- JDK 1.8
- Maven
- MySQL
- nodejs

### 技术栈
- Java
- MySQL
- SpringBoot
- MyBatis
- Vue

## 功能模块

系统主要分为以下两个角色：

### 管理员角色
- 用户管理

- 车位费用管理
- 收费管理
- 停泊管理
- 车辆进出管理

### 用户角色
- 个人中心
- 预定车位
- 缴费
- 充值
